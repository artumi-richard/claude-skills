# claude-skills

Personal Claude Code skills, kept under `~/.claude/skills/` so they're available across all
projects. Clone this repo directly into that path (or symlink it in) to pick them up on a new
machine.

## Skills

- **helpdesk-api** — interact with the helpdesk.artumi.com ticketing API (look up tickets by
  code, reply, change status/priority/worker). See `helpdesk-api/SKILL.md` for details.
